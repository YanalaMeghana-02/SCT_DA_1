# SkillCraft Technologies Data Analyst Internship

## Task 1: Excel Sales Dashboard

This repository contains the completed work for Task 1, focusing on cleaning, analyzing, and visualizing "Superstore Sales Data" using Microsoft Excel to build an interactive performance dashboard.

### Project Objectives & Features:
* **Data Cleaning & Aggregation:** Processed raw transactional data to ensure accurate metrics for analysis.
* **Pivot Table Analytics:** Built structured pivot tables to dynamically slice data by category, region, and time period.
* **Interactive Dashboard:** Designed clear visual charts highlighting **Total Sales**, **Profit by Category**, and **Sales Trends over time**.
* **Conditional Formatting:** Applied performance color scales to instantly call out top-performing segments and low-performing alerts.
